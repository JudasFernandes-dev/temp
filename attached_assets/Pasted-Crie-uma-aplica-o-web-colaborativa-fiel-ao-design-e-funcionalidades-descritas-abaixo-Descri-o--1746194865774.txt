Crie uma aplicação web colaborativa fiel ao design e funcionalidades descritas abaixo: ## Descrição Geral Uma plataforma colaborativa para gerenciamento de projetos, hackathons e networking profissional. ## Stack Tecnológica - React - TypeScript - TailwindCSS - Drizzle ORM com PostgreSQL - Wouter (para roteamento) - TanStack Query (React Query) - Shadcn/UI (componentes de interface) - Lucide React (ícones) ## Estrutura da Interface ### Layout Geral 1. Menu superior fixo (10px altura) com: - Links de navegação à esquerda (Home, Contato, Sobre) - Ícones à direita (+3, sino de notificação, foto do usuário com dropdown, toggle tema claro/escuro) - Fundo azul claro no modo claro, cinza escuro no modo escuro 2. Menu lateral esquerdo fixo (180px largura) que desce com a página: - Avatar do usuário no topo - Seção de informações pessoais (Função, Github, LinkedIn, Link) - Botão "Editar Perfil" - Links de navegação (Página Inicial, Notificações, Mensagens, Hackathons, Projetos, Equipes, Perfil, Configurações) - Seção de conquistas após Configurações: 3 selos (amarelo/troféu, verde/medalha, vermelho/escudo) - Versão no rodapé (v1.0.0) 3. Barra lateral direita com: - Seção de idiomas - Barra de progresso - Tarefas 4. Conteúdo central responsivo 5. Rodapé completo com: - Mensagem centralizada "Conectando pessoas do mesmo cosmo" - Três colunas: - SOBRE NÓS: Quem somos, Como funciona, Trabalhe conosco, FAQ - LINKS ÚTEIS: Documentação, Política de privacidade, Termos de uso - INFORMAÇÕES: Juncal 475, +11 11 42 742 01 08, Digital • AI Strategy 3D, HELLO Digital - Ícones de redes sociais centralizados - Copyright ### Funcionalidades Específicas 1. Tema claro/escuro - Ao clicar no ícone de lua/sol, exibir modal de confirmação 2. Menu dropdown no avatar do usuário - Opções: Editar Perfil, Configurações, Sair 3. Chat - Botão fixo no canto inferior direito - Ao clicar, abre modal com lista de conversas - Ao clicar em uma conversa, abre pop-up de chat individual 4. Filtros centralizados - Barra de pesquisa e categorias centralizadas 5. Carrosséis - Três tipos com cores distintas: - Projetos: azul - Hackathons: preto - Grupos: verde - Navegação manual com setas ## Comportamento Responsivo - Menu lateral colapsa em telas menores - Carrosséis adaptam-se ao tamanho da tela - Layout flexível mantém elementos organizados ## Cores e Estilo - Usar system de cores Tailwind + paleta personalizada: - Projetos: azul (#3B82F6) - Hackathons: preto/cinza escuro (#111827) - Grupos: verde (#10B981) - Conquistas: amarelo (#FCD34D), verde (#10B981), vermelho (#EF4444) - Sombras sutis nos cards e elementos flutuantes - Bordas arredondadas nos elementos interativos - Transições suaves ## Conteúdo Demonstrativo - Dados de exemplo para projetos, hackathons e grupos - Mensagens de chat de amostra - Progresso e tarefas aleatórias ## Considerações Técnicas - Usar sistema de tema via contexto React - Armazenar preferências de tema no localStorage - Implementar todos os modais como portals para evitar problemas de z-index - Carregar dados iniciais via React Query - Garantir performance com otimizações de re-renderização
